公开 认证入口服务 API

轻量认证入口，负责凭证校验与 JWT 签发。 API 文档匿名可见，无需认证即可浏览。

  • POST /auth/login — 管理员登录，返回 JWT
  • GET /auth/me — 验证令牌并返回当前用户
  • GET /healthz — 健康检查

PostgREST 业务 API

空域约束、飞行障碍、航迹规划等业务 API。 需要 role=admin JWT 才能看到完整接口。 匿名用户看到的是空白 OpenAPI 文档——这是 PostgREST 按角色公开 API 的正常行为，不要误判为 PostgREST 故障。

PostgREST OpenAPI 当前将按匿名角色加载；粘贴 admin JWT 后会使用该令牌重新请求完整业务 API 文档。

提示：从上方输入栏粘贴登录后获得的 access_token 后点击"应用令牌"， PostgREST OpenAPI 将加载完整的业务接口。令牌存储在浏览器 localStorage 的 postgrest.jwt 键中， 与前端工作台共享同一令牌。
