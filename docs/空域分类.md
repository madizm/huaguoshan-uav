# 国家空域基础分类方法（Agent 友好整理版）

> 用途：将原始法规文本整理为结构化 Markdown，便于 agent 检索、抽取规则、生成表格或实现空域判定逻辑。  
> 来源：原文标题为《国家空域基础分类方法》，依据《中华人民共和国飞行基本规则》及其补充规定制定。本文不改变原意，仅做结构化整理。

## 1. 适用范围

- **适用空域**：中华人民共和国领空内空域。
- **参照执行空域**：领空外至我国飞行情报区边界空域。
- **除外区域**：香港飞行情报区、台北飞行情报区。

## 2. 分类总览

依据航空器飞行规则和性能要求、空域环境、空管服务内容等要素，空域划分为 7 类：A、B、C、D、E、G、W。

| 类别 | 管制属性 | 典型范围 | 服务概览 | 主要飞行方式 |
|---|---|---|---|---|
| A | 管制空域 | 标准气压高度 6000 米（含）至 20000 米（含） | 为所有飞行提供空中交通管制服务并配备间隔 | 通常仅仪表飞行 |
| B | 管制空域 | 民用运输机场上空 | 为所有飞行提供空中交通管制服务并配备间隔 | 仪表、目视 |
| C | 管制空域 | 建有塔台的通用航空机场上空 | 为所有飞行提供空中交通管制服务；按飞行类型提供间隔/交通信息/避让建议 | 仪表、目视 |
| D | 管制空域 | 标准气压高度高于 20000 米；或 A/B/C/G 以外按需划设 | 为所有飞行提供空中交通管制服务；为仪表-仪表配备间隔 | 仪表、目视 |
| E | 管制空域 | A/B/C/G 以外按需划设 | 仅为仪表飞行提供空中交通管制服务；尽可能提供交通信息 | 仪表、目视 |
| G | 非管制空域 | B/C 以外真高 300 米以下（W 除外）；或平均海平面高度低于 6000 米且不影响民航公共运输飞行的空域 | 仅提供飞行信息服务，不提供空中交通管制服务 | 仪表、目视 |
| W | 非管制空域 | G 类空域内真高 120 米以下的部分空域 | 原文未列服务内容 | 微型、轻型、小型无人驾驶航空器飞行 |

## 3. 结构化分类规则

### 3.1 A 类空域

```yaml
code: A
controlled: true
range:
  vertical: 标准气压高度 6000 米（含）至 20000 米（含）
service:
  - 为所有飞行提供空中交通管制服务
  - 配备间隔
flight_requirements:
  allowed_rules:
    - 通常仅允许仪表飞行
  communication:
    - 航空器和空中交通管理部门之间必须保持持续双向无线电通信
  surveillance:
    - 航空器必须安装二次雷达应答机（同等性能的监视设备）
  flight_plan:
    - 飞行计划经过审批
    - 航空器进入前须获得空中交通管理部门许可
  pilot_qualification:
    - 航空器驾驶员应具备仪表飞行能力及相应资质
```

```yaml
code: B
controlled: true
range:

          height: 跑道道面至机场标高 600 米（含）

    - airport_type: 民用单跑道机场
      rings:
        - radius: 12 千米
          height: 跑道道面至机场标高 600 米（含）
service:
  - 为所有飞行提供空中交通管制服务
  - 配备间隔
flight_requirements:
  allowed_rules:

    - 航空器必须安装二次雷达应答机（同等性能的监视设备）
  flight_plan:
    - 飞行计划经过审批
    - 航空器进入前须获得空中交通管理部门许可
  pilot_qualification:
    - 航空器驾驶员应具备仪表或目视飞行能力及相应资质
```

### 3.3 C 类空域

```yaml
code: C
controlled: true
range:
  location: 建有塔台的通用航空机场上空
  structure:
    radius: 5 千米
    height: 跑道道面至机场标高 600 米（含）
service:
  - 为所有飞行提供空中交通管制服务
  - 为仪表和仪表飞行之间配备间隔
  - 为仪表和目视飞行之间配备间隔
  - 为目视和目视飞行之间提供交通信息
  - 根据要求为目视和目视飞行之间提供交通避让建议
flight_requirements:
  allowed_rules:
    - 仪表飞行
    - 目视飞行
  speed_limit:
    condition: 平均海平面高度 3000 米以下的目视飞行
    max_indicated_airspeed: 450 千米/小时
  communication:
    - 航空器和空中交通管理部门之间必须保持持续双向无线电通信
  surveillance:
    - 航空器必须安装二次雷达应答机或其他可被监视的设备
  flight_plan:
    - 飞行计划经过审批
    - 航空器进入前须获得空中交通管理部门许可
  pilot_qualification:
    - 航空器驾驶员应具备仪表或目视飞行能力及相应资质
```

### 3.4 D 类空域

```yaml
code: D
controlled: true
range:
  - 标准气压高度高于 20000 米
  - A、B、C、G 类空域以外，可根据运行需求和安全要求选择划设为 D 类
service:
  - 为所有飞行提供空中交通管制服务
  - 为仪表和仪表飞行之间配备间隔
  - 为仪表飞行提供关于目视飞行的交通信息，并根据要求提供交通避让建议
  - 为目视飞行提供关于仪表和目视飞行的交通信息，并根据要求提供交通避让建议
flight_requirements:
  common:
    allowed_rules:
      - 仪表飞行
      - 目视飞行
    speed_limit:
      condition: 平均海平面高度 3000 米以下
      max_indicated_airspeed: 450 千米/小时
    surveillance:
      - 平均海平面高度 3000 米以上飞行必须安装二次雷达应答机（同等性能的监视设备）
      - 平均海平面高度低于 3000 米飞行安装其他可被监视的设备
    flight_plan:
      - 必须报备飞行计划
    pilot_qualification:
      - 航空器驾驶员应具备仪表或目视飞行能力及相应资质
  special:
    entry_permission:
      - 仪表飞行航空器进入前须获得空中交通管理部门许可
      - 目视飞行航空器进入前须获得空中交通管理部门许可
    communication:
      - 必须保持持续双向无线电通信
```

```yaml
  - 为仪表飞行尽可能提供关于目视飞行的交通信息
  - 为目视飞行尽可能提供关于仪表和目视飞行的交通信息
flight_requirements:
  common:
    allowed_rules:
      - 仪表飞行
      - 目视飞行
    speed_limit:
      condition: 平均海平面高度 3000 米以下
      max_indicated_airspeed: 450 千米/小时
    surveillance:
      - 平均海平面高度 3000 米以上飞行必须安装二次雷达应答机（同等性能的监视设备）
      - 平均海平面高度低于 3000 米飞行安装其他可被监视的设备
    flight_plan:
      - 必须报备飞行计划
    pilot_qualification:
      - 航空器驾驶员应具备仪表或目视飞行能力及相应资质
  special:
    instrument_flight:
      entry_permission: 进入前须获得空中交通管理部门许可
      communication: 保持持续双向无线电通信
    visual_flight:
      entry_permission: 不需要空中交通管理部门许可
      reporting: 进入前必须报告
      listening_watch: 在规定通讯频率上保持守听
```

```yaml
  surveillance:
    - 航空器必须安装或携带可被监视的设备
  flight_plan:
    - 必须报备飞行计划
  pilot_qualification:
    - 航空器驾驶员应具备仪表或目视飞行能力及相应资质
```

## 4. 通用要求与例外

### 4.1 通信与监视覆盖

- A、B、C、D、E 类空域应当实现通信和监视覆盖。
- G 类空域应当实现监视覆盖。

### 4.2 特别批准例外

经空中交通管理部门特别批准：

- 航空器可按照目视飞行规则在 A 类空域飞行。
- 航空器可超过限制速度在 C、D、E、G 类空域飞行。

### 4.3 隔离空域

- 难以满足飞行要求时，航空用户可申请划设隔离空域并对外公布。

### 4.4 B、C 类空域边界调整

- B、C 类空域范围可根据实际情况进行调整。
- B、C 类空域可描述为不规则的多边形。

### 4.5 有人驾驶与无人驾驶适用规则

- A、B、C、D、E、G 类空域明确的飞行要求适用有人驾驶航空器。
- 无人驾驶航空器进入上述空域，按照《无人驾驶航空器飞行管理暂行条例》明确的要求执行。

### 4.6 目视飞行气象条件

| 高度条件 | 能见度 | 距云水平距离 | 距云垂直距离 / 云外要求 |
|---|---:|---:|---|
| 平均海平面高度 3000 米以上 | 不小于 8 千米 | 不小于 1500 米 | 垂直距离不小于 300 米 |
| 平均海平面高度 900 米或真高 300 米两者取较高值，至平均海平面高度 3000 米 | 不小于 5 千米 | 不小于 1500 米 | 垂直距离不小于 300 米 |
| 平均海平面高度 900 米以下或真高 300 米以下两者取较高值 | 不小于 5 千米 | — | 云外飞行 |

### 4.7 特殊任务类飞行

- 特殊任务类飞行按照起降机场开放条件和执飞机组起降标准执行。

## 5. Agent 判定提示

### 5.1 推荐判定顺序

1. **先判定特殊机场空域**：民用运输机场上空优先检查 B 类；建有塔台的通用航空机场上空检查 C 类。
2. **再判定高度层**：标准气压高度 6000 米（含）至 20000 米（含）通常为 A 类；高于 20000 米为 D 类。
3. **再判定低空非管制空域**：B、C 类以外真高 300 米以下通常为 G 类，其中 G 类内真高 120 米以下的部分空域可能为 W 类。
4. **剩余空域按需划设**：A、B、C、G 类以外，根据运行需求和安全要求选择 D 或 E 类。
5. **无人机另行适用条例**：A/B/C/D/E/G 的有人驾驶飞行要求不能直接替代无人驾驶航空器管理要求。

### 5.2 关键字段解释

- **标准气压高度**：按标准气压基准计算的高度。
- **平均海平面高度**：相对于平均海平面的高度。
- **机场标高**：机场基准高程。
- **真高**：相对于地面或障碍物的实际高度。
- **跑道道面**：以跑道表面作为下界的高度基准。

## 6. 名词解释

| 术语 | 解释 |
|---|---|
| 飞行信息服务 | 向飞行中的航空器提供有益于安全和有效地实施飞行的建议和信息的服务。 |
| 交通信息 | 由空中交通服务单位发出的信息，告诫航空器驾驶员可能在其位置或预定航线附近存在其他已知的或可以观察到的空中交通，以协助航空器驾驶员避免相撞。 |
| 目视飞行规则 | 按照目视气象条件飞行的管理规则。 |
| 仪表飞行规则 | 按照仪表气象条件飞行的管理规则。 |
| 管制空域 | 对按仪表飞行规则和目视飞行规则飞行的航空器提供空中交通管制服务而划定的空间。 |
| 非管制空域 | 不对航空器提供空中交通管制服务的空间。 |
| 交通避让建议 | 为协助航空器驾驶员避免碰撞，由空中交通服务单位提供的进行机动飞行的建议。 |
| 空中交通管制服务 | 为防止航空器之间以及机动区内航空器和障碍物之间相撞，并加速和维持有秩序的空中交通而提供的服务。 |
